Capabilities

Designed for Polimi Students

No more navigating back and forth across overlapping tables. Organize your study session schedule cleanly and securely.

Interactive Calendar Grids

View your exam dates mapped out into beautiful month, week, or day calendars. Integrates seamlessly with FullCalendar directly inside the exam registration portal.

Dynamic Color-Coded Legend

Each exam course is automatically assigned a color. Check and uncheck course filters in a clean, integrated legend layout to hide/show specific exams in real-time.

One-Click ICS Export

Instantly export individual exam schedules or the entire calendar to standard .ics format. Fits Google Calendar, Apple Calendar, and Outlook.

Integrated Countdowns

Use the action popup to scroll straight to the exam section or open a live, real-time countdown to your exam on The Anxious Display helper page.

Quick Start

Install the Extension

Available for Firefox, Chrome, Edge, and other major Chromium-based browsers.

Verified Add-on Store

Firefox is our primary distribution platform. Install the official version directly from the Firefox Add-ons store with single-click installation and automatic updates.

Install from Firefox Add-ons
01

1. Download Release

Download the compiled extension bundle zip file from our latest GitHub releases page, then extract it to a directory on your machine.

Go to Releases
02

2. Enable Dev Mode

Open your Chrome browser and navigate to chrome://extensions/. Locate and toggle the Developer mode switch in the upper-right corner.

03

3. Load Unpacked

Click on the Load unpacked button in the top left, select the extracted dist/ folder from step 1, and the extension will immediately activate!

Frequently Asked Questions

Publishing an extension on the Chrome Web Store requires creating a developer account and paying a registration fee. Since this project is a free utility built for Polimi students, we distribute it for free on Firefox Add-ons (which has no publication fees) and make it available for manual installation on Chrome, Edge, and other Chromium browsers.

No, absolutely not. The extension runs entirely inside your local browser. It reads the exam dates from the page's HTML structure and renders the calendar view locally. No network requests are made, and no credentials, names, or exam calendars are ever tracked or sent to any external server.

When you click the "Export ICS" button in the calendar toolbar, the extension reads all the rendered exam dates and generates a standardized .ics calendar file. You can import this file directly into Google Calendar, Apple Calendar, Microsoft Outlook, or any other calendar application of your choice.
